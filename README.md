# Prolog Labs — Logic Programming

Academic laboratory assignments for the **Logic Programming** course at Yachay Tech University.

## Topics Covered

- **Family trees & relationships** — predicates, rules, ancestor/sibling inference
- **Recursive algorithms** — factorial, list operations
- **List processing** — length, append, sum using recursion
- **Knowledge bases** — facts, rules, and queries in Prolog
- **Predicate logic** — unification and backtracking
